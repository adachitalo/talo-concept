# TALO — Concept Site

**TERVE TALO — 健康な木の家**

TALOインターナショナルのブランドコンセプトサイトのプロトタイプ。フィンランド発祥のマシンカットログハウスを日本の風土に合わせて進化させてきたTALOの哲学・技術・約束を伝えるためのWebデザイン。

## Live

https://adachitalo.github.io/talo-concept/

## Design

- **Typography** — Cormorant Garamond（英語見出し）、Noto Serif JP（日本語）、DM Sans（ラベル）
- **Color** — フォレストグリーン × ウッドブラウン × クリームのアースカラー
- **Tone** — 北欧 × 日本。静かだが芯がある
- **Responsive** — デスクトップ / タブレット / スマホ対応。年配ユーザーを考慮したフォントサイズ

## Key Concepts

- **TERVE TALO** — フィンランド語で「健康な家」。TALOのブランドコンセプト
- **LOG on LOG** — 総ログの英語表現。ログの上にログを積む。1階から2階まで妥協なく
- **建物は環境の一部であり、人間は自然の一部である** — ページ全体を貫く哲学的前提
